1.vue项目的创建
2.vue的启动过程

创建过程如下:

创建项目：

vue init webpack projectName（项目名）
新手推荐不要按张eslint(否则会从入门到放弃)
? Use ESLint to lint your code? (Y/n)  选择n

进入项目，下载依赖：

npm install 或者 cnpm install

运行项目：

npm run dev

cnpm是淘宝镜像,可以提高安装速度,淘宝镜像的安装方法

npm install -g cnpm --registry=https://registry.npm.taobao.org

到此为vue.js的初始化工作已完成,启动后会看到Welcome to Your Vue.js App

启动过程

个人的理解不一定准确:
vue的整个页面都会挂在到index.html id=app的div的模块中,把这个div看成是一张纸，每个vue的组建都会挂在到这里进行显示(不妨把id进行修改发现页面内容会消失),App.vue是一个全局占位点里面最核心的就是
相当于一个占位符,每一组会占一个，所以相当于组建首先挂在到App.vue的上，然后挂在到index.html的app的div中，main.js是一个全局插件注册点，还有很多细节,具体开发时候可以体会。
